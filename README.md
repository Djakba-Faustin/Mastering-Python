# Mastering Python

A beginner-friendly, hands-on learning notebook that teaches core Python concepts with examples and exercises. This repository contains a Jupyter Notebook (`learnin.ipynb`) that walks through installation, basic syntax, data types, control flow, functions, object-oriented programming, and other fundamentals. Intended for learning and practice — work in progress.

## Contents

- `learnin.ipynb` — Main learning notebook with chapters and runnable examples.
  - 1. Introduction to Python (why learn Python, installation, running scripts, IDEs)
  - 2. Data Structures & Operators (numbers, strings, lists, tuples, sets, dicts, bytes, None)
  - 3. Loops & Control Structures (for, while, break/continue/pass, if/elif/else)
  - 4. Functions (built-ins, user-defined, lambda, recursion)
  - 5. Introduction to OOP (classes, objects, methods, attributes)
  - 6. Advanced OOP (inheritance, polymorphism, encapsulation, decorators)
  - Virtual environments, running notebooks, and many small code examples

Note: The notebook contains an initial Table of Contents; some table content is still being completed. This README will be updated as the notebook grows.

## Quick start

1. Clone the repo:

   ```bash
   git clone https://github.com/Djakba-Faustin/Mastering-Python.git
   ```

2. Create and activate a virtual environment (recommended):

   - macOS / Linux:

     ```bash
     python -m venv .venv
     source .venv/bin/activate
     ```

   - Windows (PowerShell):

     ```powershell
     python -m venv .venv
     .\.venv\Scripts\Activate.ps1
     ```

3. Install Jupyter (optional if already installed):

   ```bash
   pip install jupyter
   ```

4. Open the notebook:

   ```bash
   jupyter notebook
   # or
   jupyter lab
   ```

5. In the browser, open `learnin.ipynb` and run the cells to follow along.

If you prefer not to run locally, GitHub will render the notebook so you can read it online.

## What you'll learn (high level)

- How to set up Python and a virtual environment
- Basic syntax and I/O (print, input)
- Core data types and collections: int, float, complex, str, list, tuple, set, dict, bytes, bytearray, memoryview, None
- Type conversion and common operators
- Indexing, slicing, and common built-in methods for strings, lists, and dicts
- Control flow: if-statements, loops, and loop control statements
- Functions, lambdas, and recursion
- Basic and advanced OOP concepts and examples
